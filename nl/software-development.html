Voorkom lock-in

Standaarden maken software beter. De RESTful linked data API's van onze software hanteren W3C standaarden, zodat hergebruikers eenvoudig met jouw data aan de slag kunnen. Ook bij het schrijven van de code zelf hanteren we conventies en veelgebruikte technologie, zodat jouw ontwikkelteam eventueel zelf verder kunnen met onze code. Zo blijf je niet van ons afhankelijk.

Technologie op enterprise niveau

Benieuwd naar ons gereedschap? De kern van een back-end bouwen wij op Ruby on Rails, gepaard met ons APEX framework. Wij hanteren een microservice architectuur om flexibiliteit en herbruikbaarheid te waarborgen. Onze software wordt geleverd in docker images, zodat schalen zo eenvoudig mogelijk is. We ontwikkelen snelle, gebruiksvriendelijke front-ends in React. Om de ontwikkelsnelheid zo hoog mogelijk te houden en fouten te voorkomen hanteren we strikte ontwikkelmethoden.
